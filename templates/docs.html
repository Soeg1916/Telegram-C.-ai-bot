Getting Started

Welcome to Character Chat Bot! Follow these simple steps to begin chatting with your favorite characters:

  1. Open Character Chat Bot in Telegram.
  2. Send the /start command to initiate the bot.
  3. Use the /characters command to see a list of available characters.
  4. Select a character by sending /character [character_id].
  5. Start chatting with your chosen character!
Tip: Use the /help command at any time to see a list of available commands.

Bot Commands

The Character Chat Bot offers the following commands:

Command Description
/start Initialize the bot and receive a welcome message.
/help Display a list of available commands and their descriptions.
/characters Show a list of all available characters.
/character [id] Select a character to chat with or view your current character if no ID is provided.
/reset Reset your conversation with the current character.
/stats Show stats for your current character, including mood and conversation count.
/create Start the process of creating a custom character.
/delete [id] Delete a custom character that you've created.

Character Profiles

Each character in our bot has a unique personality profile that determines how they respond to your messages.

Personality Traits

Characters have five personality traits, each rated on a scale of 1-10:

  • Friendliness: How warm and approachable the character is.
  • Humor: How likely the character is to use jokes or sarcasm.
  • Wisdom: How knowledgeable and insightful the character is.
  • Patience: How tolerant the character is of repetitive or confusing questions.
  • Emotional: How strongly the character expresses emotions.
Mood System

Characters have a dynamic mood system that changes based on your interactions:

  • All characters start with a neutral mood (5/10).
  • Positive and respectful conversations improve a character's mood.
  • Negative or rude messages can worsen a character's mood.
  • A character's mood affects the tone and content of their responses.
Memory

Characters remember your previous conversations (up to 30 messages) and can reference them in future chats.

Custom Characters

Want to chat with a character that's not on our list? You can create your own custom character!

Creating a Custom Character
  1. Send the /create command to start the character creation process.
  2. Follow the prompts to provide:
    • A name for your character
    • A brief description
    • Personality trait values
    • A system prompt (optional, for advanced users)
  3. Once completed, your character will be added to your personal character list.
Managing Custom Characters
  • Use /characters to see your custom characters along with preset characters.
  • Custom characters are private and only visible to you.
  • Use /delete [id] to remove a custom character you've created.
Note: Custom characters must adhere to our content guidelines. Characters that promote harmful content are not allowed.

API Reference

Character Chat Bot provides a simple REST API for accessing character data:

GET /api/characters

Returns a list of all preset characters.

Response Format:
{
  "characters": {
    "character_id": {
      "name": "Character Name",
      "description": "Character description",
      "traits": {
        "friendliness": 7,
        "humor": 5,
        "wisdom": 8,
        "patience": 6,
        "emotional": 4
      }
    },
    // More characters...
  }
}
GET /api/character/{character_id}

Returns details for a specific character.

Path Parameters:
  • character_id - The ID of the character to retrieve
Response Format:
{
  "name": "Character Name",
  "description": "Character description",
  "traits": {
    "friendliness": 7,
    "humor": 5,
    "wisdom": 8,
    "patience": 6,
    "emotional": 4
  },
  "background": "Character background information",
  "sample_phrases": [
    "Hello there!",
    "How are you today?",
    "Tell me more about yourself."
  ]
}
