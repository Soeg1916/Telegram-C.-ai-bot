Getting Started

To start using Character Chat Bot, simply message our bot on Telegram at @your_bot_username.

Basic Commands

Here are the essential commands to get started:

/start - Start the bot and get a welcome message
/help - Show available commands and usage information
/characters - List all available characters
/select [character_id] - Select a character to chat with

Chatting with Characters

After selecting a character using /select, you can start chatting right away. Simply send messages as you would in a normal conversation, and the character will respond.

Character Memory

Characters remember your conversation history and will refer to past interactions. Their mood and relationship with you evolve based on your conversations.

/reset - Reset your conversation with the current character
/stats - Show your relationship stats with the current character

Creating Custom Characters

You can create your own custom characters with unique personalities, traits, and backstories.

Character Creation Process

/create - Start the character creation process

The bot will guide you through the creation process, asking for:

  1. Character name
  2. Description/background
  3. Personality traits
  4. System prompt (how the character should act)
  5. NSFW setting (whether the character can engage in mature content)

Managing Custom Characters

/mycharacters - List characters you've created
/delete [character_id] - Delete a custom character you've created

Sharing Characters

You can share your custom characters with the community through our approval system.

/share [character_id] - Request to share a custom character with the community
/publiccharacters - Browse community-shared characters

NSFW Mode

For adult users, characters can engage in more mature and uninhibited conversations.

/togglensfw - Toggle NSFW mode for the current character

Note: NSFW content is only available for users who are 18+. Please use this feature responsibly and in accordance with Telegram's policies.

API Reference

Our bot offers a simple API for developers to integrate with. Here are the available endpoints:

Endpoint Method Description
/api/characters GET Get a list of all available characters
/api/character/{character_id} GET Get details for a specific character
/api/status GET Check if the API is running

Frequently Asked Questions

How do I change characters?

Use the /select [character_id] command to switch to a different character. You can find character IDs with the /characters command.

Why did my character's mood change?

Characters' moods change based on your interactions. Be kind and considerate to improve their mood, or intentionally argumentative to lower it. You can check their current mood with /stats.

Can I use this bot in group chats?

Yes, but the bot will only respond when mentioned or replied to in group chats.
